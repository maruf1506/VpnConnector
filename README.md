# VpnConnector
VpnKey Server Connector

Add it in your root build.gradle at the end of repositories:

```
  allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
```
  
Step 2. Add the dependency

```
	dependencies {
	        implementation 'com.github.maruf1506:VpnConnector:1.2'
	}
 ```
